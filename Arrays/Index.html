1. Allocate the array

Write a script that allocates array of 20 integers and initializes each element by its index multiplied by 5. Print the obtained array on the console.



2. Compare two arrays

Write a script that compares two char arrays lexicographically (letter by letter).




3. Max sequence

Write a script that finds the maximal sequence of equal elements in an array.Example: {2, 1, 1, 2, 3, 3, 2, 2, 2, 1} -> {2, 2, 2}.



4. Max increasing sequence

Write a script that finds the maximal increasing sequence in an array. Example: {3, 2, 3, 4, 2,2, 4} -> {2, 3, 4}.
